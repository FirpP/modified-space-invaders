{\footnotesize\ttfamily int kbhit (\begin{DoxyParamCaption}\item[{void}]{ }\end{DoxyParamCaption})}



Sprawdza w sposób nieblokujący, czy naciśnięto klawisz klawiatury. 

Funkcja ta emuluje zachowanie znanej z systemu Windows funkcji \+\_\+kbhit() w środowiskach zgodnych z POSIX (Linux, mac\+OS). Pozwala sprawdzić stan bufora wejścia bez zatrzymywania działania programu.

Działanie funkcji opiera się na tymczasowej modyfikacji ustawień terminala\+:
\begin{DoxyEnumerate}
\item Pobiera obecne ustawienia terminala (struktura termios).
\item Wyłącza tryb kanoniczny (ICANON), aby znaki były dostępne natychmiast bez konieczności wciskania Enter.
\item Wyłącza echo (ECHO), aby testowany znak nie pojawiał się na ekranie.
\item Ustawia deskryptor wejścia (STDIN) w tryb nieblokujący (O\+\_\+\+NONBLOCK), dzięki czemu próba odczytu nie zatrzyma programu, jeśli bufor jest pusty.
\item Próbuje pobrać jeden znak funkcją getchar().
\item Natychmiast przywraca oryginalne ustawienia terminala.
\item Jeśli znak został pobrany, jest zwracany do bufora (ungetc), aby kolejna funkcja (np. get\+Key) mogła go poprawnie odczytać.
\end{DoxyEnumerate}

\begin{DoxyReturn}{Returns}
int
\begin{DoxyItemize}
\item 1 (true)\+: Jeśli w buforze wejścia znajduje się znak (klawisz został wciśnięty).
\item 0 (false)\+: Jeśli bufor jest pusty (brak wciśniętego klawisza).
\end{DoxyItemize}
\end{DoxyReturn}
